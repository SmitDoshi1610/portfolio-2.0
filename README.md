# Smit Doshi | Modern Developer Portfolio

A premium, highly interactive, and SEO-optimized portfolio website built with **Next.js**, **TypeScript**, and **Vanilla CSS**. This project showcases my journey as a Software Engineer, my technical expertise, and my professional achievements with a focus on high-end aesthetics and performance.

## 🚀 Live Demo
[Check out the live portfolio here](https://smitdoshi1610.github.io/Portfolio/)

## ✨ Key Features

- **Premium Design System**: A custom-built CSS design system featuring glassmorphism, smooth gradients, and mesh background effects.
- **Dynamic Theming**: Support for both **Dark** and **Light** modes with a persistent toggle (remembers your choice via `localStorage`).
- **Advanced Core Web Vitals**: Optimized for fast loading and zero Cumulative Layout Shift (CLS) using `next/font`.
- **High-End Interactivity**:
    - **Particle Network**: A real-time canvas-based particle network in the Hero section.
    - **Magnetic Buttons**: Buttons that subtly track the mouse pointer for a tactile feel.
    - **Custom Cursor**: A smooth trailing dual-circle cursor system.
    - **Typewriter Effect**: Dynamic introduction of roles.
    - **Scroll Reveal**: Sections glide into view smoothly using the `Intersection Observer API`.
- **SEO & Social Optimization**:
    - **JSON-LD Structured Data**: Helps search engines index professional details accurately.
    - **Metadata API**: Robust Open Graph and Twitter Card support for premium social sharing previews.
    - **Sitemap & Robots**: Pre-configured for optimal crawling.
- **Professional Layouts**:
    - **LinkedIn-Style Experience**: A clean, industry-standard presentation of my career timeline.
    - **Project Showcase**: Visual cards with unique accent glows for diverse projects.
    - **Coding Profiles**: Integrated links to GFG, LeetCode, CodeChef, Codeforces, and HackerRank.

## 🛠️ Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: Vanilla CSS (Custom Variables & Modern Layouts)
- **Deployment**: Optimized for Vercel / GitHub Pages
- **Icons**: Lucide & Custom SVGs

## 📦 Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/SmitDoshi1610/Portfolio.git
   ```

2. **Navigate to the project directory**:
   ```bash
   cd Portfolio
   ```

3. **Install dependencies**:
   ```bash
   npm install
   ```

4. **Run the development server**:
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 🏗️ Project Structure

```text
/app             # Next.js App Router (Pages & Layout)
/components      # Reusable UI Components (Hero, About, etc.)
/public          # Static assets (Images, Resume, Favicon)
/globals.css     # Global styles & Theme variables
```
